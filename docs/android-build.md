# Android Build

Open `RemoteControl/android` in Android Studio.

## Requirements

- Android Studio
- Android SDK 35
- Java 17

## Build

1. Sync Gradle.
2. Run the app on an emulator or device.
3. In Settings, set the backend URL.

For an Android emulator talking to a local backend, use:

```text
http://10.0.2.2:5000
```

For production, use the HTTPS Render URL.
